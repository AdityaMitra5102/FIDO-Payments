FIDO Payments is a seamless and fast web-based framework for leveraging digital lending processes to sustainably serve unbanked or underserved segments with (digital) credit. It uses principles of device attestation to eliminate complications in loan application and simplifies the entire lending process. This framework can be used both in online as well as offline banking.

What if the user loses the FIDO2 key?

He/She/They can login via Email OTP to assign new FIDO2 key.

When the user is unable to log in but needs someone else to?

Since FIDO2 is a Passwordless technology, the family members of the user can use the security key.

What are the system requirements for the user?

Any android phone with Google Chrome 89+ and NFC reader.
